# 🎮 Qutqaruvchi

![Qutqaruvchi](image.png)

**Qutqaruvchi** — 3D shooter o'yini.

## 📖 Hikoya

Sun'iy intellekt (AI) dunyoni egalladi. Endi insoniyatning taqdiri sizning qo'lingizda.

Qurollaringiz bilan AI qo'shinlariga qarshi kurashing, dushmanlarni yo'q qiling va dunyoni qutqaring.

## ✨ Xususiyatlari

* 🔫 3D shooter gameplay
* 🤖 AI dushmanlari
* 🎵 Ovoz effektlari
* 🌆 Futuristik muhit
* 💾 Rekord saqlash tizimi
* 🇺🇿 To'liq o'zbek tilida

## 📥 O'rnatish

1. Repository yuqori qismidagi **Qutqaruvchi Installer.exe** faylini yuklab oling.
2. Installer'ni ishga tushiring.
3. O'yinni o'rnating.
4. Ish stolidagi **Qutqaruvchi** yorlig'i orqali o'yinni ishga tushiring.
